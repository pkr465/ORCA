Powerful Capabilities

ORCA combines static analysis, AI-powered insights, and automated remediation to ensure your codebase meets compliance standards.

📊

Static Analysis

7 comprehensive analyzers with 41+ rules for code style, whitespace, macros, and structural validation.

🧠

Context-Aware Analysis

Pre-analyzes code structure via header resolution and auto-generated constraints to enrich LLM prompts and reduce false positives.

🤖

LLM Semantic Analysis

Claude and GPT-powered semantic understanding enriched with codebase context for deeper compliance insights. Supports Anthropic, OpenAI, and QGenie providers.

⚙️

HITL Fixer Workflow

3-mode repair orchestrator: Excel-driven fixer, single-patch analysis, and batch-patch processing with automated remediation, backup, and diff generation.

📜

License Compliance

Automatic SPDX, REUSE, and copyright detection with comprehensive license header validation across 50+ licenses.

📋

Multi-Format Reports

Generate detailed compliance reports in Excel, JSON, HTML, and custom formats with HITL feedback persistence via PostgreSQL.

Quick Start

Get ORCA up and running in minutes with these simple commands.

chmod +x install.sh && ./install.sh
./launch.sh
./launch.sh --website
python main.py audit --codebase-path /path/to/code --enable-context
python main.py context-gen --codebase-path /path/to/code
python main.py fixer-workflow --excel-file out/detailed_code_review.xlsx
